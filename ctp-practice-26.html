Anatomy of Importing

Modules give us ready-made tools. Click each part below to understand how we bring in (`import`) and use a function (`sqrt`) from the `math` module.

import math
math.sqrt(16) # Calculate square root

Identify all 8 parts!

The `import` keyword: Tells Python to load a module. Module name (`math`): The specific collection of tools we want. Using the module name (`math`): We specify *which* module's tool we need. The dot (`.`): Accesses something *inside* the module. Function name (`sqrt`): The specific tool (function) from the `math` module. Opening parenthesis `(`: Starts the input(s) for the function. Argument (`16`): The value the `sqrt` function will work on. Closing parenthesis `)`: Ends the input(s) for the function.
