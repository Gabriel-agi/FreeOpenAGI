Anatomy of a Function Definition

Functions bundle code. Click each highlighted part of this simple function definition to learn its role. This uses Abstraction to name a task.

def greet(): print("Hello!")

Identify all 5 parts!

The `def` keyword: Signals the start of a function definition. Function Name (`greet`): Your chosen name for this block of code. Follows variable naming rules. Parentheses `()`: Required after the name. Holds input parameters (arguments) later. Empty for now. Colon `:`: Marks the end of the function header and start of the indented body. Function Body (indented): The code that runs when the function is *called*. Indentation is crucial!
