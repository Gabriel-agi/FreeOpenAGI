Anatomy of a Function Definition Abstraction

Functions bundle code. Let's dissect how we define one. Click each highlighted part of the function definition below.

def greet(name):
    # Code block goes here (indented)

Identify all 6 parts!

`def` keyword: Signals the start of a function definition. `greet`: The chosen name for this function (follows naming rules). Opening parenthesis `(`: Starts the parameter list. `name`: A parameter - a placeholder for data the function will receive. Closing parenthesis `)`: Ends the parameter list. Colon `:`: Marks the end of the function header and the start of its code block.
