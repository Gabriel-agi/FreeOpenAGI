Anatomy of a `while` Loop

A `while` loop repeats code as long as a condition is true. Click the parts below to learn about them.

count = 0 # Initialize a counter
while count < 3: print(count)
count = count + 1 # IMPORTANT: Update the counter!

Identify all 4 parts!

The `while` keyword: Signals the start of a conditional loop. The condition (`count < 3`): A True/False test. The loop continues ONLY if this is True. The colon `:`: Marks the end of the `while` statement and the beginning of the indented block. The loop body (indented block): Code executed repeatedly as long as the condition is True. MUST be indented.

Trace the Loop

Read the code below carefully. What will be the final value printed? Type your prediction and check your answer.

x = 10
while x > 5:
print(x) x = x - 2 # Loop finishes, then this line runs
print("Final x:", x)

Explanation:

The loop starts with `x = 10`. It continues as long as `x > 5`. Iteration 1: `x` is 10 (prints 10), `x` becomes 8. Iteration 2: `x` is 8 (prints 8), `x` becomes 6. Iteration 3: `x` is 6 (prints 6), `x` becomes 4. Now `x` is 4, which is NOT greater than 5, so the condition `x > 5` is False. The loop stops. The final `print` statement outside the loop runs with the current value of `x`, which is 4. So the output is "Final x: 4".
