Anatomy of a Print Statement

Let's break down that first piece of code. Click on each highlighted part below to identify it.

print("AI is cool!" ) # Display a message

Identify all 5 parts!

The `print` function: Python's command to show output. Opening parenthesis `(`: Starts the list of things for the function to use. A "string": Text data, always wrapped in quotes (`""` or `''`). Closing parenthesis `)`: Ends the list of things for the function. A comment `#`: A note for humans; Python ignores this.
