Anatomy of a `for` Loop

Loops help us repeat code. Click on each highlighted part of this `for` loop to understand its role.

for i in range(3): ----print("Looping!")

Identify all parts!

The `for` keyword: Starts the loop definition. Loop variable (`i`): Holds the current item/number from the sequence in each iteration. The `in` keyword: Connects the variable to the sequence it will loop over. `range(3)`: Creates a sequence of numbers (0, 1, 2) for the loop to iterate through. Colon `:`: Marks the end of the `for` statement line and indicates the start of the loop body. Indentation (Space): REQUIRED! Defines the block of code (body) that gets repeated. Loop Body (`print(...)`): The code that is executed repeatedly in each iteration.
