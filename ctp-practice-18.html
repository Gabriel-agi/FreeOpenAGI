Let's practice sending values back with the `return` statement!

Spotlight on `return`

Functions can send back results using `return`. Click the highlighted parts of this simple function to see what they do. Notice how this supports Abstraction.

def add_numbers(num1, num2):
   return num1 + num2

Identify all parts!

`def`: Keyword to define a new function. `add_numbers`: The name we give to our function. `(`: Starts the list of inputs (parameters) the function accepts. `num1`: The first input value (parameter) the function receives. `,`: Separates multiple parameters. `num2`: The second input value (parameter). `)`: Ends the parameter list. `:`: Marks the end of the function definition line (header). `return`: Keyword that sends a value *back* from the function. `num1 + num2`: The expression whose result is calculated and sent back.

Predict the Returned Value

When the code below runs, what value will the function calculate_area *return*? Type your answer in the box.

def calculate_area(length, width):
