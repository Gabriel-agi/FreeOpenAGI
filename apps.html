AGI Companion Browser

AI Agent Browser for Android

Open Source / Available

Control your Android web browsing visually using AI! AGI Companion Browser acts as an AI Agent, allowing advanced multimodal models (starting with Gemini, with Qwen, DeepSeek, Llama planned) to understand web pages via screenshots and interact using precise grid-based commands. Navigate, click, type, and automate tasks just by describing your goal. Explore the future of web interaction!

Key Features:

  • AI-powered visual web control (Gemini, others planned)
  • Grid-based targeting for clicks & typing
  • Multi-turn task execution & context memory via notes
  • Full browser features (tabs, desktop mode, downloads)
  • Configurable API Key support
  • Open Source (MIT Licensed)
