AGI Companion Browser

AI Agent Browser for Android

Open Source / Available

Control your Android web browsing visually using AI! AGI Companion Browser acts as an AI Agent, allowing advanced multimodal models (starting with Gemini, with Qwen, DeepSeek, Llama planned) to understand web pages via screenshots and interact using precise grid-based commands. Navigate, click, type, and automate tasks just by describing your goal. Explore the future of web interaction!

Key Features:

  • AI-powered visual web control (Gemini, others planned)
  • Grid-based targeting for clicks & typing
  • Multi-turn task execution & context memory via notes
  • Full browser features (tabs, desktop mode, downloads)
  • Configurable API Key support
  • Open Source (MIT Licensed)

Arcane Tome Chat

Themed AI Chat Interface

Available

Engage with AI models through a highly customizable, themed chat interface. Arcane Tome supports multiple API providers and models, persistent chat history, custom system prompts ("Guidance"), and various visual themes. Uniquely, settings and API/chat management menus are accessed by clicking the spirit (bot) and user profile pictures (sigils) respectively, providing a clean and immersive experience.

Key Features:

  • Multiple visual themes (Fantasy, Cyberpunk, etc.)
  • Multi-language support
  • Support for various API providers & models (configurable)
  • Saved chat history (via IndexedDB)
  • Customizable system prompts ("Spirit Guidance")
  • Avatar customization
  • Markdown rendering for AI responses
  • Unique avatar-click menu access
