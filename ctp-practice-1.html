Anatomy of Variable Assignment

Variables store data. Click each part of the assignment statement below to understand its role. This uses Abstraction (giving data a name) and Data Representation (storing info).

user_name = "Alice" # Store the user's name

Identify all 4 parts!

The Variable Name: A label (abstraction) for a memory location. Choose meaningful names! The Assignment Operator `=`: Assigns the value on the right to the variable on the left. The Value: The actual data being stored (represented). Here, it's a string. A Comment `#`: Explains the code's purpose; ignored by Python.
