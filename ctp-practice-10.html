Anatomy of Lists

Lists group items together. Click the parts below to see what they do. Lists are fundamental Data Structures.

tech_items = ["CPU", "GPU", "RAM"]
print(tech_items[1]) # Accessing the second item

Identify all parts!

Variable Name: A label to store the list. Assignment `=`: Puts the list into the variable. Opening Bracket `[`: Starts the list definition. List Element: An item stored in the list (here, strings). Comma `,`: Separates elements in the list. Closing Bracket `]`: Ends the list definition. `print` function: Displays output. Opening parenthesis `(`: Starts function arguments. List Variable: Using the list's name. Opening Bracket `[`: Starts index access. Index Number: Specifies *which* element (starts at 0!). Closing Bracket `]`: Ends index access. Closing parenthesis `)`: Ends function arguments. Comment `#`: Note for humans.
