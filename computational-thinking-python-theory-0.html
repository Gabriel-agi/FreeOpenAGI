✨ Your First Glimpse: "Hello, World!"

Let's see a tiny piece of Python code. It's a classic tradition!

print("Hello, AI World!") # This line tells Python to display text

This single line of code instructs the computer to display the message "Hello, AI World!" on the screen.

print() is a function – a pre-defined action.

The text inside the quotes "Hello, AI World!" is the data we want to show. Text data needs quotes!

The part starting with # is a comment - a note for humans that the computer ignores.
