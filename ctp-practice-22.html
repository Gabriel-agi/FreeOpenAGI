Anatomy of Tuple Creation & Access

Tuples are defined with parentheses `()` and items separated by commas. They are ordered, and you access items using square brackets `[]`. Click the parts below. 

point = (10, 25, "Origin")

x_coord = point[0] # Access first element

Identify all parts!

Variable name: Stores the tuple. Assignment operator `=`: Assigns the value (tuple) to the variable. Opening parenthesis `(`: Starts the tuple definition. Element: An item inside the tuple (number, string, etc.). Comma `,`: Separates elements within the tuple. Closing parenthesis `)`: Ends the tuple definition. Variable name: Stores the accessed element. Tuple name: The variable holding the tuple being accessed. Opening square bracket `[`: Starts the index access. Index `0`: The position of the element to access (starts from 0). Closing square bracket `]`: Ends the index access. Comment `#`: Note for humans, ignored by Python.
