Identify all operators!

`==` (Equal): Checks if two values are the same. `!=` (Not Equal): Checks if two values are different. `<` (Less Than): Checks if the left value is smaller than the right. `>` (Greater Than): Checks if the left value is larger than the right. `<=` (Less Than or Equal): Checks if the left value is smaller than or equal to the right. `>=` (Greater Than or Equal): Checks if the left value is larger than or equal to the right.
