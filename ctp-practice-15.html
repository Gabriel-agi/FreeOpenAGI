Anatomy of Loop Control

Loops sometimes need finer control. Click on the highlighted keywords (`for`, `if`, `break`, `continue`) to understand their roles in this example.

for num in range(1, 6):
    if num == 3:
        continue # Skip number 3
    if num == 5:
        break # Stop when 5 is reached
    print(num)

Identify all 4 parts!

The `for` loop: Repeats a block of code for each item in a sequence (here, numbers 1 to 5). The `if` statement: Checks a condition. Code inside only runs if the condition is True. `continue`: Skips the rest of the current loop iteration and jumps to the next one. `break`: Immediately exits the entire loop, regardless of remaining items or conditions.
