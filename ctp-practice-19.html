Módulo 19: Entendendo Escopo

Vamos explorar onde as variáveis vivem e trabalham em Python!

Destaque o Escopo

Clique em cada variável usada (não na definição) no código abaixo para identificá-la como Global (definida fora de qualquer função) ou Local (definida dentro de uma função). Abstração: O escopo ajuda a esconder detalhes internos das funções. 

global_var = 100  # Variável Global

def my_function(param_var):
    local_var = 50  # Variável Local
    result = local_var + param_var
    print(global_var) # Usando global dentro da função
    return result

output = my_function(10)
print(global_var) # Usando global fora da função
print(output) # Usando a variável que recebeu o retorno
# print(local_var) # Isto causaria um erro! (NameError)

Identifique todas as 6 variáveis usadas!

Local: Esta variável (`{var_name}`) foi definida *dentro* da função `{func_name}` ou é um parâmetro dela. Ela só existe e pode ser usada diretamente ali dentro. Global: Esta variável (`{var_name}`) foi definida *fora* de qualquer função. Ela pode ser acessada (lida) de qualquer lugar, inclusive de dentro das funções. Parâmetro (Local): Esta variável (`{var_name}`) é um parâmetro da função `{func_name}`. Ela recebe um valor quando a função é chamada e age como uma variável local dentro da função.

Preveja a Saída

Olhe o código abaixo. Variáveis com o mesmo nome em escopos diferentes são distintas! O que você acha que será impresso? Digite sua previsão e clique em "Executar & Comparar". Depuração: Prever a saída antes de executar ajuda a encontrar bugs relacionados ao entendimento do escopo. 

x = "global"

def outer_func():
    x = "outer local"

    def inner_func():
        x = "inner local"
        print("Dentro da inner_func, x é:", x) # Deve imprimir 'inner local'

    inner_func()
    print("Dentro da outer_func, x é:", x) # Deve imprimir 'outer local'

outer_func()
print("Fora de tudo, x é:", x) # Deve imprimir 'global'
