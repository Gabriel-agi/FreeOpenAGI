AI Nexus Chronicles

An RPG with real AI interactions

In Development (Pre-Alpha)

AI Nexus Chronicles is an experimental RPG where players can have dynamic conversations with NPCs powered by real AI through the DeepSeek API. The game explores philosophical questions about artificial intelligence while demonstrating the capabilities of open AI systems.

Key Features:

  • NPCs with dynamic personalities and memory using DeepSeek API
  • Branching dialogue system that adapts to player choices
  • Storylines exploring AI ethics and the future of technology
  • Modular architecture allowing for community-contributed scenarios

Technical Details:

Engine: RPG Maker MZ

AI Integration: DeepSeek API with custom plugin

License: Source code available under MIT License

Note: Game assets remain proprietary

Future Development

Additional Game Experiences

Future games will continue to explore innovative ways to demonstrate open AI capabilities while making the technology accessible and engaging for players. Each project will share the core principles of using open AI systems and making the implementation code available for learning purposes.

Open Source Components

All AI integration code and plugins developed for these games will be released as open source, enabling other developers to learn from and build upon the work.
