Trace with `break`

The `break` statement exits a loop immediately. Predict the exact output of this code, including all printed lines.

i = 0
while i < 10:
print("Current i:", i) if i == 3:
print("Breaking!") break i = i + 1 print("Loop finished.")

Explanation:

The loop starts with `i = 0`. - `i=0`: Prints "Current i: 0". `i` becomes 1. - `i=1`: Prints "Current i: 1". `i` becomes 2. - `i=2`: Prints "Current i: 2". `i` becomes 3. - `i=3`: Prints "Current i: 3". The condition `i == 3` is True. Prints "Breaking!". The `break` statement is executed, immediately stopping the loop. The `i = i + 1` line inside the loop is skipped. - The code after the loop runs, printing "Loop finished.". The final output consists of these lines exactly.
