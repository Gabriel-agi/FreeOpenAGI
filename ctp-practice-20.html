Anatomy of a Dictionary

Dictionaries store data in key-value pairs. Click each highlighted part of this example dictionary to learn its role.

student = {'name': 'Alice', 'age': 25}

Identify all 9 parts!

Opening Brace `{`: Marks the start of a dictionary. Key `'name'`: A unique identifier (usually a string) for a value. Colon `:`: Separates a key from its value. Value `'Alice'`: The data associated with the key 'name' (a string here). Comma `,`: Separates key-value pairs. Key `'age'`: Another unique identifier. Colon `:`: Separates the 'age' key from its value. Value `25`: The data for 'age' (a number here). Closing Brace `}`: Marks the end of the dictionary.
